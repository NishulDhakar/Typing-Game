## 🛠️ Quick Start

```bash
# Clone this typing paradise
git clone https://github.com/NishulDhakar/Typing-Game.git

# Enter the arena
cd Typing-Game

# Equip your weapons (dependencies)
npm install

# Start your training
npm run dev
```

## 📁 Project Structure

```bash
typing-game/
├── 📱 src/                  # Source files
│   ├── 🧩 components/       # UI components
│   │   ├── Button.jsx        # Interactive buttons
│   │   ├── Footer.jsx        # Page footer
│   │   ├── LoadingScreen.jsx # Loading animation
│   │   ├── Stats.jsx         # Performance metrics
│   │   ├── ThemeToggle.jsx   # Dark/Light switch
│   │   └── TypingArea.jsx    # Main game area
│   ├── ⚙️ utils/           # Helper functions
│   ├── 🎨 styles/          # CSS styles
│   └── 📄 App.jsx          # Main component
└── 📦 public/              # Static files
```

## 🎮 Key Features

### ⌨️ Typing Challenge
- Practice with curated text samples
- Real-time error highlighting
- Instant feedback system

### 📊 Performance Analytics
- Words Per Minute (WPM) tracking
- Accuracy percentage
- Keystroke analysis
- Progress over time graphs

### 🌓 Customization
- Dark/Light theme toggle
- Adjustable difficulty levels
- Custom text input option

- Mobile-friendly interface

- Adaptive layout
